# AI_spam_email_detector
A Machine Learning-based web application that classifies emails/messages as Spam or Not Spam (Ham). This project uses Natural Language Processing (NLP) techniques and a trained model to detect unwanted messages efficiently.


🚀 Features
📩 Detects whether an email/message is spam or not
🧠 Uses Machine Learning for prediction
🔤 Text preprocessing using NLP techniques
⚡ Fast and interactive UI using Streamlit
📊 Real-time prediction output



🛠️ Tech Stack
Python
Scikit-learn
Pandas & NumPy
NLTK (Natural Language Processing)
Streamlit (Frontend UI)
